Sanity-check units¶

units=metric was requested explicitly from the API (see 00_download.ipynb). NYC TMAX should be roughly -15..40 °C; if the raw GHCN tenths-of-degree convention had leaked through instead, values would look like ~arbitrary integers in the hundreds (e.g. 250 instead of 25.0).
